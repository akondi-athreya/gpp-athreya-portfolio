## Project Description

This is a fully responsive, animated personal portfolio website built from scratch using React and Vite.
It presents professional background, technical skills, featured projects, and contact details in a polished single-page experience.

## Tech Stack

- React
- Vite
- Framer Motion
- CSS3
- React Icons

## Local Setup and Run

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Enter the project folder:
   ```bash
   cd athreya-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```
6. Preview production build locally:
   ```bash
   npm run preview
   ```
